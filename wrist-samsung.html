Project Overview

Wearable Blood Pressure Monitoring

At Samsung Research America, I developed a clinical validation algorithm combining statistical modeling and machine learning (MATLAB) to analyze digital biomarkers for wellness detection in healthcare wearable devices.

I designed and executed a validation study protocol using an oscillometric reference standard to establish ground truth for continuous blood pressure monitoring algorithm development — bridging clinical rigor with consumer wearable technology.

Key Contributions

Validation & Analysis

I conducted root cause analysis to identify data synchronization failures in a large-scale clinical study and provided analytical support for implementing data integrity solutions. I also developed a sensor validation protocol using correlation analysis to assess ambient light sensor accuracy for health monitoring applications.
